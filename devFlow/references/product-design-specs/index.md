# 产品设计规范索引

本目录用于存放产品给出的字段、校验、展示和交互规范。使用时先读本索引，再根据页面涉及的字段类型读取对应分册。不要一次性加载全部规范。

## 读取流程

1. 先识别页面中涉及的字段类型、列表能力、详情展示、表单提交、筛选、上传、选择器、弹窗和操作反馈。
2. 先读取
   `field-common.md`，它是字段布局、必填、校验、错误提示、只读、列表展示的通用规范。
3. 根据字段类型读取对应分册。
4. 页面涉及默认值、空状态、操作反馈、弹窗、文案或中断恢复时，读取
   `interaction.md`。
5. 如果字段类型无法判断，不要套用规范，写入待确认项。

## 输出要求

在页面技术方案中引用这些规范时：

- 不要整段复制规范正文。
- 只提炼和当前页面相关的约束。
- 如果页面字段与规范冲突，优先写入“风险与待确认项”。
- 如果规范要求产品确认某项内容，例如小数位数、数据来源、人员范围，应明确列为待确认问题。
