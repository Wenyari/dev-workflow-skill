# ID / 流水号 / 编码规范

## 适用范围

- ID：合同ID、补充协议ID。
- 流水号：订单号、交易流水、凭证号。
- 编码：服务编码、员工工号、地区代码、国家代码。

## 录入方式

- 系统自动生成，不可编辑。
- 无特殊说明时，在数据提交时生成。
- 生成格式需要引用编码规则。

## 字段说明

- 提示文本：系统自动生成。
- 帮助文本：简单描述生成格式和示例。

## 字段校验

- 无前端字段校验。

## 列表显示

- 默认隐藏，按需显示。
- 筛选：模糊查询。
- 高级筛选：等于、不等于、包含、不包含、为空、不为空。
- 排序：升序或降序；字母部分按文本排序，数字部分按数值大小排序。

## 产品要点

- 必须说明生成格式和示例。
- 如需前端显示，必须特别说明显示位置。

## UI 与体验要点

- 如果前端可见，列表和详情页内需要支持一键复制，并有复制成功反馈。

## 研发要点

- ID 冲突时自动加 1，需要重试机制，确保唯一。
- 流水号冲突时自动加 1，需要重试机制，确保唯一。
- 编码需要重试机制，确保生成成功，并支持重置。
