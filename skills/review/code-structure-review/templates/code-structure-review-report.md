## 检测限制

- Graphify 可能漏掉动态导入、反射、依赖注入、模板引用和运行时注册。
- 邻居集合相似不代表实现、控制流或业务语义相同。
- 社区划分来自图拓扑，不等于真实模块边界。
- 第一版未执行 AST 子树克隆检测。
- 第一版未分析 Git 共同变更历史。
