# 文本字段规范

## 短文本

### 适用范围

- 标题、名称、身份证号、邮箱号码、手机号码、固定电话等。

### 录入方式

- 文本输入、扫码输入、语音输入。
- 默认值：无、自动计算、固定文本。
- 通用输入限制：
  - 禁止空格、换行。
  - 长度限制：20 或 100 字符，具体按需求确认。
- 特殊输入限制：
  - 手机号码（中国大陆）：等于 11 位，仅允许数字字符。
  - 固定电话：不含分机符号部分不超过 15 位；仅允许数字字符、连字符、分机标识 `#`
    或 `ext`。
  - 手机号（全球）：仅过滤明显非法字符，例如字母、表情符号。
  - 邮箱：小于等于 100 字符。
  - 身份证号（中国大陆）：等于 18 位，仅允许数字字符和 `X`。
  - 统一社会信用代码：`^[159]\d[0-9]{6}[0-9A-Z]{9}[0-9A-Z]$`
  - 组织机构代码：`^[0-9A-HJ-NP-RT-UW-Y]{8}[0-9X]$`

### 字段说明

- 提示文本：请输入。
- 帮助文本：一般无帮助文本，如有要求按需说明。

### 字段校验

- 手机号码（中国大陆）：`^1(3\d|4[5-9]|5[0-35-9]|6[2567]|7[0-8]|8\d|9[0-35-9])\d{8}$`
- 固定电话：`^0\d{2,3}-?\d{7,8}(#\d{1,6})?$`
- 身份证号（中国大陆）：`^[1-9]\d{5}(18|19|20)?\d{2}(0[1-9]|1[0-2])(0[1-9]|[12]\d|3[01])\d{3}[\dXx]$`
- 身份证号（中国大陆）必须实现 GB 11643-1999 校验码验证，不能只靠正则。

### 列表显示

- 显示：按需显示。
- 筛选：模糊查询。
- 高级筛选：等于、不等于、包含、不包含、为空、不为空。
- 排序：按字母升序或降序。

### UI 与体验要点

- 邮箱号码可支持自动补全常用域名。
- 手机号（全球）支持国际区号下拉选择，默认 `+86`，或根据 IP、浏览器语言推荐。

## 长文本

### 适用范围

- 备注、说明、简介等。

### 录入方式

- 手动输入，支持空格、换行。
- 支持语音输入。
- 默认值：无。
- 无特殊说明时，默认限制 200 或 2000 字符，超出禁止输入。

### 字段说明

- 提示文本：请输入。
- 帮助文本：一般无帮助文本，如有要求按需说明。

### 字段校验

- 无。

### 列表显示

- 一般不支持显示，如有需求需特殊说明。
- 一般不支持筛选。
- 高级筛选：不支持。
- 排序：不支持。

### UI 与体验要点

- 编辑态默认 5 行，超出可滚动。
- 底部显示已输入字数和总字数。
- 详情态按内容高度自适应。

### 研发要点

- 存储时保留换行符。
- 展示时转义。
- 需要防范 XSS 注入。

## 富文本

### 适用范围

- 公告、通知、详情描述、邮件内容等。

### 录入方式

- 富文本编辑器。
- 基础能力：缩进、对齐、加粗、斜体、下划线、有序列表、无序列表、图片、超链接。
- 高级能力：表格、清除格式、全屏、快捷键。
- 默认值：无；如有模版规范，可按需设置模版。

### 字段说明

- 无。

### 字段校验

- 无。

### 列表显示

- 不支持显示。

### 产品要点

- 需要明确是否支持模版。
- 需要明确是否需要 @ 人员。
- 需要明确是否支持附件插入。

### UI 与体验要点

- 工具栏保持简洁，默认收起部分高级功能。

### 研发要点

- 输出 HTML 或 JSON 格式。
- 需要做 XSS 过滤。
- 图片上传需要支持压缩与附件存储。
